# 1. QWidget的概念

## 1. QWidget的重要性
- QWidget是Qt中所有用户界面对象的基类
- QWidget的窗看不见接受鼠标键盘等事件
- 可以通过屏幕上绘制自己
- 父子关系有相对坐标
  >坐标: 左上角是(0,0),横着是x轴,纵轴是y轴.\
  >绝对坐标: 对象在窗口上的坐标\
  >相对坐标: 对象相对于父类的坐标.(比如当窗口移动时,子部件最好用相对坐标定位)

```## 2. QThread
创建一个线程
- 创建一个类,该线程接口集成QThread
- void run(){}//重载线程处理函数
- void start(){}//线程启动函数

## 3. 手动创建QWidget对象
- QWidget w;//创建独立窗体
- w.show();//显示包含子窗口,是槽函数,只能在主线程中调用
- w.hide();//隐藏包含子窗口,是槽函数
